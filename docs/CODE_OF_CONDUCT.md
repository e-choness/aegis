# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, inclusive, and constructive. Harassment of any kind
is not tolerated.

## Reporting

To report an issue, contact the project maintainers via GitHub Security
Advisories (for security-related conduct issues) or open a GitHub Discussion.
Reports are handled confidentially.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these
standards. Violations may result in a temporary or permanent ban.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
